# ScrollViewContext.scrollTo(Object object)

-  功能描述

滚动至指定位置

-  参数

**Object object**

| 属性     | 类型    | 默认值 | 必填 | 说明             |
| :------- | :------ | :----- | :--- | :--------------- |
| top      | number  |        | 否   | 顶部距离         |
| left     | number  |        | 否   | 左边界距离       |
| velocity | number  |        | 否   | 初始速度         |
| duration | number  |        | 否   | 滚动动画时长     |
| animated | boolean |        | 否   | 是否启用滚动动画 |
