# wx.onWifiConnected(function listener)

> 支持度：
>
> Android：支持
>
> iOS：支持

## 功能描述

监听连接上 Wi-Fi 的事件

## 参数

### function listener

连接上 Wi-Fi 的事件的监听函数

#### 参数

##### Object res

| 属性 | 类型                                                         | 说明       |
| :--- | :----------------------------------------------------------- | :--------- |
| wifi | [WifiInfo](./WifiInfo.md) | Wi-Fi 信息 |
